## 📚 需求文档详细说明

### 1. 首页 (01_首页)

**文档路径**: `01_首页/01_首页.md`

**需求场景数**: 22 个 Given-When-Then 场景

**功能概述**:
- 整体页面布局：背景为白色，分上、中、下三部分布局
- 顶部导航栏：包含logo区域、搜索框区域、特别入口区域
- 常用功能快捷入口：提供"首页"、"车票"、"团购服务"、"会员服务"、"站车服务"、"商旅服务"、"出行指南"、"信息查询"等入口
- 车票查询表单区域：包含车票查询表单，支持出发地、目的地、出发日期等查询条件

**主要功能**:
- 车票查询
- 常用功能导航
- 用户登录/注册入口

**文档路径**: `02_登录注册页/02_1_登录页.md`

**需求场景数**: 21 个 Given-When-Then 场景

**功能概述**:
- 整体页面布局：背景为白色，分上、中、下三部分布局
- 顶部导航栏：与首页一致的导航栏布局
- 登录表单区域：包含账号登录选项、用户名/邮箱/手机号输入框、密码输入框、立即登录按钮
- 登录辅助功能：提供"注册12306账户"和"忘记密码？"入口

**主要功能**:
- 账号登录
- 密码找回
- 注册跳转

### 3. 注册页 (02_2_注册页)

**文档路径**: `02_登录注册页/02_2_注册页.md`

**需求场景数**: 35 个 Given-When-Then 场景

**功能概述**:
- 整体页面布局：与首页一致，背景为白色，分上、中、下三部分布局
- 账户信息表单：包含用户名输入框、登录密码输入框、确认密码输入框、证件类型选择框、姓名输入框、证件号码输入框、优惠类型选择框、邮箱输入框、手机号码输入框、服务条款勾选框、"下一步"按钮

**主要功能**:
- 用户注册
- 证件类型选择（8种证件类型）
- 优惠类型选择（成人、儿童、学生、残疾军人）
- 服务条款确认

### 4. 订单填写页 (03_订单填写页)

**文档路径**: `03_订单填写页/03_订单填写页.md`

**需求场景数**: 14 个 Given-When-Then 场景

**功能概述**:
- 整体页面布局：与首页一致，背景为白色，分上、中、下三部分布局
- 列车信息窗口：显示车次日期、列车号、出发站、到达站、出发时间、到达时间、一等座/二等座/硬卧/硬座的剩余中位数与价格
- 乘客信息窗口：包含乘客信息区域和车票信息区域
- 选项按钮：提供"上一步"和"提交订单"按钮

**主要功能**:
- 列车信息展示
- 乘客信息选择
- 车票类型选择
- 订单提交

### 5. 车次列表页 (04_车次列表页)

**文档路径**: `04_车次列表页/04_车次列表页.md`

**需求场景数**: 7 个 Given-When-Then 场景 + 9 个 页面间 Given-When-Then 场景

**功能概述**:
- 页面从上到下分为：顶部导航栏、车次查询栏、车次详细信息选择区、车次信息展示区、底部导航栏五个部分
- 车次查询栏：支持单程/双程选择，出发地、目的地、出发日、返程日（如双程则可选）选择
- 车次详细信息选择区：提供车次筛选功能
- 车次信息展示区：展示符合条件的车次列表，包含车次号、出发时间、到达时间、历时、商务座、一等座、二等座、硬卧、硬座、无座等席位信息

**主要功能**:
- 车次查询
- 车次筛选
- 车次信息展示
- 车次选择

### 6. 个人中心页 (05_个人中心页)

**文档路径**: `05_个人中心页/05_个人中心页.md`

**需求场景数**: 22 个 页面内 Given-When-Then 场景 + 11 个 页面间 Given-When-Then 场景

**功能概述**:
- 页面从上到下分为：顶部导航栏、个人中心主页、底部导航栏三个部分
- 个人中心主页具有左侧栏，从上到下的加粗板块为"个人中心"、"订票中心"、"本人车票"、"会员中心"、"个人信息"、"常用信息管理"、"温馨提醒"、"投诉和建议"
- 顶部导航栏：包含logo、搜索栏、功能按钮组
- 个人中心主页：显示用户真实名称、问候语、会员服务和车票服务接口、微信和支付宝二维码等信息接收方式设置

**主要功能模块**:
- **个人中心主页**: 用户信息展示、问候语、服务入口
- **火车票订单**: 未完成订单、未出行订单、历史订单管理
- **查看个人信息**: 基本信息、联系方式、优惠类型查看和编辑
- **乘车人**: 乘车人信息管理（添加、编辑、删除乘车人）

**子功能**:
- 订单查询和筛选
- 订单状态管理（未完成、未出行、历史）
- 个人信息管理
- 乘车人管理
- 信息接收方式设置
